To finish setup, configure the Grafana plugin:

  1. Install the Neutrino app plugin into Grafana
  2. Set the following environment variable on your Grafana deployment:

       NEUTRINO_BACKEND_URL=http://{{ include "neutrino.fullname" . }}.{{ .Release.Namespace }}.svc.cluster.local:{{ .Values.service.port }}

  3. Enable the plugin in Grafana → Administration → Plugins → Neutrino

Once enabled, Neutrino appears in the Add dropdown in Grafana Explore.

    wget -qO- http://localhost:{{ .Values.service.port }}/readyz
